# Screenshots Directory

This directory is for storing screenshots of the Tools page for the README carousel.

## Recommended Screenshots

1. **chain-builder.png** - Chain Builder canvas showing states and transitions
2. **simulation.png** - Simulation in action with highlighted paths
3. **grammar-editor.png** - Grammar Editor with example grammars
4. **analysis.png** - Analysis tab showing transition matrices and properties
5. **string-testing.png** - String acceptance testing visualization

## How to Add Screenshots

1. Take screenshots of your Tools page (1920x1080 or similar resolution)
2. Save them in this directory with descriptive names
3. Update the README.md carousel section with the image paths

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Optimize images (use tools like TinyPNG)
- **Naming**: Use kebab-case (e.g., `chain-builder.png`)
- **Alt Text**: Always include descriptive alt text for accessibility
